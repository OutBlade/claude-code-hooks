# Contributing to Claude Code Hooks

Thank you for your interest in contributing to the Claude Code Hooks project! We welcome contributions from everyone.

## Guidelines for Contributing
1. **Identify the Area:** Before contributing, check if the area you want to work on is already being handled by someone else. 
2. **Fork the Repo:** Start by forking the repository to your GitHub account.
3. **Create a Branch:** Create a new branch for your feature or bug fix. It's good practice to name your branch descriptively.

   ```bash
   git checkout -b your-feature-name
   ```

4. **Write Clear Commits:** Write clear commit messages that explain your changes.

## Development Setup
To get started with development, follow these steps:
1. **Clone the Repository:** Clone your forked repository to your local machine:
   ```bash
   git clone https://github.com/OutBlade/claude-code-hooks.git
   ```
   Replace `OutBlade` with your GitHub username if necessary.

2. **Install Dependencies:** Navigate to the project directory and install any required dependencies. Check the `README.md` for specific instructions.

3. **Run the Project:** Ensure that you can run the project locally before making changes. This helps verify that your development environment is set up correctly.

## Pull Request Process
1. **Submit a Pull Request:** Once you have made your changes, push your branch to your forked repository and submit a pull request to the main repository.

   ```bash
   git push origin your-feature-name
   ```

2. **Provide Context:** In your pull request description, include a description of your changes and any relevant information.
3. **Review and Feedback:** Be open to feedback and prepared to make changes to your pull request as necessary. 
4. **Merge Process:** After your pull request is approved, it will be merged into the main branch.

Thank you for contributing!
